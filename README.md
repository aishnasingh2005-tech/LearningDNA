# 🧠 Learning DNA Engine

An AI-powered Personalized Learning Recommendation System developed using Machine Learning and Flask.

## Features

- Predicts Learning Style using Machine Learning
- Personalized Study Recommendations
- Weekly Study Plan
- Download PDF Report
- Prediction History
- Search Predictions
- Delete Predictions
- Export CSV
- SQLite Database Integration

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Scikit-Learn
- Pandas
- ReportLab

## Machine Learning Algorithm

Random Forest Classifier

## Dataset

Custom student dataset containing:

- Age
- Lessons Completed
- Quiz Score
- Learning Preference

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open

http://127.0.0.1:5000
